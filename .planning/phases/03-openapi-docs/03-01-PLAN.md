```rust
pub(crate) fn make_router(state: Arc<AppState>) -> Router {
    let cors = CorsLayer::new()...;
    let db_routes = Router::new()
        .route(...)
        ...
        .with_state(state.clone());
    let rpc_routes = Router::new()
        .route(...)
        ...
        .with_state(state.clone());
    let admin = admin_routes(state.clone()).with_state(state.clone());
    db_routes
        .merge(rpc_routes)
        .nest("/admin", admin)
        .layer(cors)
        .layer(from_fn_with_state(state, track_metrics))
}
```

SwaggerUi merge pattern (from RESEARCH.md — merge AFTER final .with_state on the combined router):
```rust
// The merged router must be Router<()> before SwaggerUi can merge into it.
// The existing db_routes and rpc_routes each call .with_state(state.clone()) — they are already Router<()>.
// So the final merge chain is already Router<()> before .layer(cors).
// Insert SwaggerUi merge before the .layer() calls:
db_routes
    .merge(rpc_routes)
    .nest("/admin", admin)
    .merge(SwaggerUi::new("/swagger-ui")
        .url("/api-docs/openapi.json", ApiDoc::openapi()))
    .layer(cors)
    .layer(from_fn_with_state(state, track_metrics))
```
